# Business Portfolio

Welcome to the **Business Portfolio** repository! This project is a modern, fully responsive SaaS and business portfolio template built with HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Overview

This repository contains the source code for a high-performance business website designed to showcase services, portfolios, and company information. Originally based on the Dewi Bootstrap template, it has been heavily customized with interconnected pages, tailored professional English copy, and seamless user experiences.

## ✨ Features

- **Modern & Responsive Design:** Built on Bootstrap for a fluid, mobile-first experience across all devices.
- **Interconnected Pages:** Seamlessly navigate between the Home, Portfolio Details, Service Details, and Starter pages.
- **Tailored SaaS Copy:** All generic placeholder text has been replaced with professional, conversion-focused English copy.
- **Dynamic Interactions:** Includes smooth scrolling, sticky navigation, mobile dropdowns, and dynamic carousels.
- **SEO & Performance Optimized:** Clean HTML semantics and organized structure for better search visibility.

## 📁 Project Structure

- `index.html` - The main landing page with sections for About, Services, Portfolio, Team, and Contact.
- `portfolio-details.html` - A dedicated page for showcasing individual project case studies and deliverables.
- `service-details.html` - A detailed view of specific services offered.
- `starter-page.html` - A blank interconnected template page for easy expansion.
- `assets/` - Contains all CSS, JS, images, and vendor libraries (Bootstrap, Swiper, GLightbox, AOS, etc.).

## 🛠️ Technologies Used

- **HTML5 & CSS3**
- **JavaScript (Vanilla)**
- **Bootstrap 5.3.3**
- **AOS (Animate On Scroll)**
- **Swiper & Isotope Layout**

## 💻 Getting Started

To view or modify this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/amoney79/Business-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Business-Portfolio
   ```
3. Open `index.html` in your preferred web browser. No local development server or build step is required!
